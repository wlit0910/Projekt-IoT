## Windows Build Instructions

To build the project on Windows, you need Visual Studio 2022.

### Steps to Build

1. **Open the Solution File**:
   - Navigate to `windows/VS2022/`.
   - Open the solution file (`.sln`).
2. **Build the Solution**:
   - Build the solution by clicking `Build` -> `Build Solution`.

### Notes:
- All required DLLs for SDL2 and SDL_ttf are already included in the `windows/external` folder.
